# ms-trime

trime 同文输入法主题, inspired by [魔法少女的魔女审判](https://manosaba.com/)(manosaba).

## 背景

某一天, 监牢的幕后黑手灵光一闪, 火速抓来一名实习生, 给每个少女的手机都安排上专属输入法皮肤。

## Usage

将 `ms.trime.yaml` 以及 `fonts/`  复制到 `trime` 的配置目录下, 重新部署后即可使用。

较原仓库的更改：
1. 更改了奈叶香的配色，并设为默认主题；
2. 更改了键盘布局，模仿讯飞输入法默认布局；
3. 增加 `move`键盘；
4. 使用了 `iconfont.tff` 中的图标；
5. 增加了几组符号。
